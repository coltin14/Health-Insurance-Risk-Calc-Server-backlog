# Health Insurance Risk Calculator: Server
This project was made by Michael Analitis, Neil Mamparo, and Coltin Rogge for CPSC-44000 Software Engineering.

## Instructions
1. `git clone https://github.com/coltin14/Health-Insurance-Risk-Calc-Server-backlog.git`
2. `cd Health-Insurance-Risk-Calc-Server-backlog`
3. `npm install`
4. `npm start`
This website is also hosted on Microsoft Azure and can be accessed [here](https://health-insurance-risk-calc-server-backlog-fngae6hphbgcchd8.centralus-01.azurewebsites.net/).
